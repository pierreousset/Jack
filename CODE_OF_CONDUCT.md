# Code of Conduct

This project adheres to the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be welcoming, assume good faith. Harassment, personal attacks, and discriminatory behavior are not tolerated.

Instances of unacceptable behavior may be reported to the maintainers at pierreousset@gmail.com. All complaints will be reviewed promptly and confidentially.
